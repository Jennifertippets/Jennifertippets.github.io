Typography reference

Heading Font: Fira Sans

I choose this font for my headings because it is clear to read. Also, it is a darker and thick font.

font-family: 'Fira Sans', sans-serif;

Example: It is a beautiful day. Enjoy!

Paragraph Font: Roboto

I like this font for my paragraphs because it is clear,strong and easy to read.

font-family: 'Roboto', sans-serif;

Example: It is a beautiful day. Enjoy!
