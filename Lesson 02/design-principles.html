White Space

UX Engineer

uxengineer.com

The White Space used in this website, such as margin, padding, and letter spacing, helps the viewer read or see the page much better. It is crucial to create white space between elements on a web page. A separation between UI elements can help to indicate a relationship or distinction, depending on how much is applied. We can see good use of white space on this page because it allows us to establish hierarchy, improves comprehension, and improve focus on the viewer.

Contrast

Little book, Big Story

littlebookbigstory.com

Contrast is an essential principle on a website. Every element such as color, font, shape, and pictures play an important role on this page. A web designer must learn how to emphasize essential elements on a web page to get the audience's attention to the page. This picture is a good Contrast design example because it only alternates with a few elements. Because using too many elements on a page can confuse or distract viewers from the intended message.
